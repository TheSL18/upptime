FAQ

¿Globalists es una aplicación web?

Así es, Globalists es una herramienta 100% web que permite realizar consultas desde cualquier dispositivo que tenga acceso a internet.

¿Globalists funciona en cualquier navegador?

Si, la plataforma esta diseñada para ser utilizada sobre cualquier navegador. Para la mejor experiencia del usuario recomendamos usar Chrome, Firefox o Safari.

¿Cuántas consultas puedo realizar al mes?

El número de consultas manuales es ilimitado en todos nuestros planes de contratación. El número de consultas vía API esta determinado por el plan contratado.

Requiero un plan a la medida, ¿Globalists me lo puede ofrecer?

Si, manejamos planes a la medida para cualquiera que sea la necesidad o requerimiento. Contáctanos en nuestro correo electrónico, teléfono o WhatsApp.
