MINDS es un sistema para la prevención de lavado de dinero

El sistema lleva a cabo todos los procesos de cumplimiento comenzando por el análisis de información de prospectos, integración de datos de clientes y figuras relacionadas, declaración de perfil transaccional, asignación de nivel de riesgo por cliente, monitoreo transaccional, generación de alarmas y reportes a las autoridades. Está dividido en módulos que permiten el monitoreo por producto financiero para diferentes tipos de análisis y manejo de información.

MÓDULOS

MINDS está desarrollado alrededor de 4 módulos principales: KYC, Cliente, Server y KYE. Cada uno de estos ayuda a cumplir correctamente con la regulación mexicana.

KYE
KYE es una herramienta que funciona principalmente para manejar los registros de información de los empleados de la institución así como su riesgo de lavado de dinero.
SERVER
Es el motor principal del sistema encargado de realizar la carga de información y procesos necesarios para el análisis y prevención de lavado de dinero.
KYC
KYC es una herramienta que tiene como principal tarea administrar los datos de los clientes de la institución, permitiendo conocer el nivel de riesgo, cuentas y partes relacionadas del sujeto.
CLIENTE
Este modulo permite revisar y calificar las alarmas detectadas, generar reportes regulatorios y revisar coincidencias en listas negras. Adicionalmente, permite administrar las configuraciones de seguridad del sistema.

FUNCIONALIDADES

MINDS tiene una gran variedad de funcionalidades, entre las cuales se incluyen los requerimientos principales de un sistema de PLD:

  • Proceso KYC (Identificación del cliente) basado en los documentos e información provista a la institución por el sujeto.
  • Verificación de nombres en listas negras. (OFAC, KINGPIN, FBI, CIA, PPE etc.)
  • Asignación de nivel de riesgo de la institución (EBR).
  • Detección y generación de reportes de operaciones específicas
  • Reportes regulatorios adicionales
  • Buzón de denuncias anónimas
  • Metodología para la detección de operaciones inusualeS
  • Estándares regulatorios para sistemas automatizados.
